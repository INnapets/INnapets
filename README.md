### Projects

**[ContextForge](https://github.com/INnapets/ContextForge)** — RAG knowledge base for Microsoft Fabric pipelines

> Crawls pipeline definitions from Fabric API and stored procedures from Azure SQL. Chunks content into typed semantic units (ActionCard, IoMap, TransformDesc, ErrorPattern), embeds via OpenAI text-embedding-3-small, stores in Qdrant with payload metadata for filtered retrieval. Exposed as MCP server with 7 tools — hybrid vector + keyword search, lineage queries, error pattern lookup. Claude Desktop queries the knowledge base in natural language instead of reading raw pipeline definitions.
